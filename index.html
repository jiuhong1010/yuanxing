Python数据分析 - 第8章习题
共10题 难度: 中等
预计用时: 20分钟 已完成: 3/10
第4题 (单选题)
4/10
在Pandas中，以下哪个函数用于处理缺失值？
A
pd.merge()
B
pd.fillna()
C
pd.concat()
D
pd.groupby()
答案解析
正确答案是 B: pd.fillna()

pd.fillna() 是 Pandas 中用于填充缺失值的函数。它可以将缺失值(NaN)替换为指定的值。

其他选项的功能:
- pd.merge(): 用于合并数据框
- pd.concat(): 用于连接数据框
- pd.groupby(): 用于分组数据
